# My Projects

Here are some of the projects I've worked on. Each one taught me something new and helped me grow as a developer.

---

## 1. Event Locator App
A full-stack web application that helps users discover local events based on their location. Built with Node.js, Express, and PostgreSQL.

**Technologies:** Node.js, Express, PostgreSQL, Redis, JWT Authentication  
**Key Features:** User registration, event creation, geolocation-based search, notifications

---

## 2. African Marketplace
An e-commerce platform showcasing African products with a modern UI designed for mobile and desktop.

**Technologies:** React, Django REST Framework, PostgreSQL  
**Key Features:** Product listings, shopping cart, payment integration, admin dashboard

---

## 3. Mini Blog Platform
A lightweight blogging platform where users can create, edit, and publish articles.

**Technologies:** React, Node.js, MongoDB  
**Key Features:** CRUD operations, Markdown support, user authentication

---

## 4. Land Restoration Platform
A platform to track and report land restoration efforts across communities.

**Technologies:** Flutter, Firebase, Google Maps API  
**Key Features:** Real-time data, geolocation tracking, community reports

---

## 5. SafeBoda Foundations
A ride-hailing simulation app with core booking features modelled after SafeBoda.

**Technologies:** React, Node.js, Socket.io  
**Key Features:** Real-time trip updates, driver matching, trip history
